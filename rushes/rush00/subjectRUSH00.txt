Ejercicio: 00
rush0X
Directorio de entrega: ex00/
Archivos a entregar: main.c, ft_putchar.c, rush0X.c
Funciones autorizadas: write

• Archivos a entregar: main.c, ft_putchar.c y el rush0X.c, donde 0X corresponderá
al número del rush. Por ejemplo, rush00.c.
• Ejemplo de main.c:
main()

int
{

rush(5, 5);
return (0);
}

• Por lo tanto, deberéis escribir la función rush que reciba como parámetro dos
variables de tipo entero que se llamen respectivamente x e y.
• Vuestra función rush tendrá que mostrar en pantalla un rectángulo de x caracteres
de ancho e y caracteres de alto.
• Vuestra función no debe producir un crash ni entrar en bucle infinito.
• Vuestro main será modificado durante la evaluación para poder cambiar los parámetros de llamada a la función rush. Por ejemplo, se probarán este tipo de cosas:
int
{

main()
rush(123, 42);
return (0);

}
